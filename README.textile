h1. Список кукбуков и рецептов

* nginx - устанавливает nginx, обеспечивает автоматическую конфигурация и активацию сайтов в соответствии с функциональными группами через атрибуты, контроль за содержанием всех директорий с конфигурационными файлами.
* php5-fpm - устанавливает php5-fpm, предоставляет возможность настройки через атрибуты, экспортирует доступные сокеты через атрибуты.
